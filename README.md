# 2023_ransomware_-public_sample-

toy sample of Federated learning for malware detection based on AI

the NDA was a very ambicious SAAS focus on user endpoints and Server hardening based on CIS Benchmarks recommendations

As part of tasks were evaluated 
  * sql and no sql options for data storage 
  * several architectures (as microservices and message brokers)
  * programming languages as python, golang, flutter, c++
  * Tools as ansible, docker and terraform
  * cloud providers as aws, azure
  * different knowledge sources
  * Several textbooks and courses
  * Several approaches for metrics and evaluation of performance
  * Several investment sources
  * Several endpoint platforms (windows/linux/android)

As output were obtained
  * thesis for master degree on english
  * several MVP prototypes
  * several investing raising applications

base team two seniors one focus on databases and other on Machine learning and software development and four juniors each
